支持的平台

以下的平台是Electron支持的

  • macOS
    只支持macOS64进制的系统，支持的最低版本是macOS 10.9

  • Windows

    支持windows7以后的系统，老系统暂不支持，并且不会运行，32位和64位都支持，ARM版本的windows
    暂不支持

  • Linux

    预编译的Electron的ia32(i686)和x64(amd64)二进制文件都是在Ubuntu12.04上编译的。
    一下的linux系统证实了可以运行预编译版本的Electron

    • Ubuntu 12.04 and later
    • Fedora 21
    • Debian 8
