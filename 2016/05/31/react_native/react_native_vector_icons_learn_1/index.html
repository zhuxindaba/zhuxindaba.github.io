如何使用自己的iconfont呢?

首先将你的定义的iconfont.css放到你的RN项目根目录下，然后在根目录下执行
./node_modules/.bin/generate-icon ./iconfont.css --componentName=MyIcon --fontFamily=myicon > MyIcon.js
