git的常用指令(一)

2016-06-16 Comments Git

  • 配置别名
    git config alias.st 'status'
    git st 的作用等于 git status,只对当前仓库有效，如果想配置全局则git config -g alias.st 'status'

  • git rm
    git rm –cached会删除索引中的文件并把它保留到工作目录中,而git rm则会将文件从索引和工作目录中都删除。

  • gitk命令查看提交图
